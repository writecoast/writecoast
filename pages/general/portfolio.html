API guides

At eCal I wrote an online API guide for their web-based calendar product, which allowed customers to integrate the eCal calendar into their own websites. This kind of integration is common in today's Internet API ecology, but in 2001 it was very cutting-edge. If you are familiar with documentation for today's modern REST API's, you can probably recognize all the same basic features in this 2001 guide: endpoint reference, "how-to" topics, sample code, and even a sandbox for submitting code to a live server.

Note: This is an archived HTML capture of a PHP-based website and is no longer hosted on its original host, so some of the links no longer work. I did not code the PHP or design the navigation for the site. My role was to write the content as HTML files and XML code samples and provide it to the site developer. The current corporate owner of the ecal.com domain is not related to my employer described here.

Online help

I learned how to create online help starting with Windows 3.1, before RoboHelp was invented (or at least before it was well-known). Back then we used Microsoft Word to create specially marked-up RTF files, and then compiled them with the Windows help compiler. Later I would use HATs such as Doc-To-Help and RoboHelp. Some of those help files are shown below. These are WinHelp files (.hlp) and do not run on modern PCs. School technology

In 2011 Delaware was planning to build a new system for education data. From the RFP, I was able to determine that the metrics planned for the system did not address the needs of students or their familes. As the kickoff post for the blog, I wrote a three-part article laying out the case on the planned system and the entities that developed the technology.

Trip to Nikola Tesla's laboratory

August 6, 2013 - Last weekend I made the trip along with my teenage son to western Long Island, to join a volunteer work crew cleaning up the site of Nikola Tesla’s laboratory, Wardenclyffe, with the intention of restoring it and creating a Tesla science museum. I couldn’t resist the opportunity for urban exploring and the chance to rub elbows with history. To round out our geeky weekend, the next day we visited the nearby Brookhaven National Laboratory and got an up-close tour of RHIC – the Relativistic Heavy Ion Collider.
